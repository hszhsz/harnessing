# 导言 一根缰绳的两千万年

## 一个深夜，一个程序员，和一头看不见的牛

凌晨两点，旧金山某栋公寓的窗户里还亮着灯。一个程序员把一段需求贴进对话框，按下回车，然后离开电脑去倒了一杯水。

他没有写代码。他对一个 AI Agent 说：把这个项目的后端迁移到新的数据库，跑通所有测试，提交一个 Pull Request，附上变更说明。

二十分钟后，他端着水杯回来。屏幕上滚动着上百条日志——文件被读取、被修改、被重写；测试一个一个跑过；commit 被打包；一封整齐的 PR 描述躺在 GitHub 上等他 review。这件事，按照三年前的工作流程，需要他和团队投入整整一周。

他在屏幕前愣了几秒钟，忽然想起一个完全不相干的画面：几千年前的两河流域，某个农民第一次把一根绳索套在野牛的脖子上。那头牛踉跄、挣扎、嘶吼，然后被人按住、被驯服、被套上犁——人类第一次让另一种生物的肌肉替自己干活。那一天，那个农民大概也曾在田边愣几秒钟，想：这是真的吗？我刚刚做了什么？

这两个场景之间，隔着两百万年。但它们其实是同一件事的不同版本。

## 为什么是现在写这本书

我们正站在一个奇怪的时间点上。

一方面，一个全新的物种正在出现：不是硅基生命那种科幻意义上的"物种"，而是一种全新的"可以被驾驭的力量"——它会写代码、会查资料、会调用其他工具、会自己判断下一步该做什么。它没有身体，但它有"行动力"。我们还没有想好该怎么称呼它，暂且叫它 Agent。

另一方面，每个人——从最资深的工程师到刚毕业的大学生——心里都隐隐有一种不安：这一次的变化和过去的不一样吗？我的工作还在吗？我的孩子十年后要学什么？我们到底在见证一场技术升级，还是一场文明级别的转折？

这种不安源于一种"无坐标感"。当一个事物正在发生，而你既没有参照系也没有地图时，你只能凭直觉去判断它的方向，而直觉在这个尺度上往往是错的。

这本书想做的事情很简单：给你一张地图。

不是一张关于未来的地图——未来没有地图，未来只有方向感。这是一张关于过去的地图，一张关于"人类每一次让外部力量替自己干活"的全景图。当你看清了从石器到蒸汽机、从印刷术到互联网这条主线之后，AI Agent 在地图上的位置就会自动浮现出来。你不会再觉得它是一道凭空出现的闪电，你会看见它是一条延续了两百万年的河流，在我们这一代人脚下汇成了一个新的入海口。

## 这本书写给谁

这本书不是写给 AI 研究员的，也不是写给历史学家的。

它写给那些会在深夜想这些问题的人：为什么我会比我祖父更轻松地养活一家人？为什么我十岁的孩子用平板的方式让我感到既熟悉又陌生？为什么我看新闻里说"AI 将改变一切"时，既兴奋又有点害怕？

它写给那些对历史好奇但不想读教科书、对技术好奇但不想读论文、对未来好奇但厌倦了空洞预言的普通读者。

阅读这本书你不需要任何前置知识。你不需要懂神经网络，不需要记住任何朝代的年号，不需要会任何编程语言。你只需要带上一点耐心，和一种愿意被故事带着走的好奇心。书里出现的每一个名词——熵、马力、晶体管、Transformer——都会在登场时被解释清楚，就像第一次见面时握手自我介绍。

如果你读完之后，能在饭桌上对家人讲一个关于"瓦特、爱迪生、和 ChatGPT 其实是同一个故事的三个章节"的故事，那这本书的使命就完成了。

## 读完之后你会得到什么

三样东西。

**第一，一条贯穿两百万年的主线。** 从奥杜威石器开始，人类做的每一件大事，本质上都是同一件事：把"必须由我自己肌肉/大脑/时间去完成的事"交出去——交给火、交给牛、交给水车、交给蒸汽机、交给电流、交给计算机、交给互联网、交给 AI。这条主线一旦看清，你看任何一个时代的关键发明，都会自动看见它在这条线上的位置。

**第二，一套理解任何新技术的框架。** 当下一个新东西出现——可能是脑机接口，可能是可控核聚变，可能是某种我们今天连名字都还没有的东西——你会有一套属于自己的判断方法：它驾驭的是什么？它把什么交出去？它解放了什么、又制造了什么新问题？这套框架不会让你预测未来，但它会让你不再被未来吓到。

**第三，一种历史定位感。** 知道自己站在哪里，是一种很大的安慰。当你意识到，从第一根缰绳到 AI Agent，人类一直在做同一件事——只是每一代人都以为自己这一代是最特别的——你会获得一种平静。这种平静不是麻木，而是一种"看清楚了之后"的清醒。

## 一个埋下的伏笔

这本书有一个英文母题：**Harness**。

它在英文里既是一个动词——驾驭、套上、利用；也是一个名词——挽具、缰绳、那条把动物和犁连在一起的皮带。今天，在 AI 行业里，它还有第三层含义：Agent Harness，指的是那一整套让模型能调用工具、记忆、自我规划的框架。

一个词，三层意思，正好覆盖了人类生产力史的开端、中段和此刻。中文里，我们把这本书译作《驾驭》。但"驾驭"只翻译了它的动作，没有翻译它的物件。这个双关意味着什么，我把它留到全书的最后一章再揭晓。在那之前，请你带着这个小小的谜题往下读。

## 一个必要的声明：隐喻的边界

在继续之前，有一件事值得说清楚。

"驾驭"这个隐喻最好用的地方，是那些存在一种明确的外在力量、而人类发明了某种接口或控制机制去引导它的场景——火、牛、蒸汽、电流、算力，都属于这一类。力量在那里，人类造出了缰绳。模式清晰，隐喻精确。

但这本书里有一些章节，"驾驭"更像是一种弱类比而非精确描述。文字的发明——你很难说人类"驾驭"了什么外在力量，更像是人类创造了一种全新的东西。印刷术、市场制度、公司这些发明也是如此：它们与其说是套住了一匹野马，不如说是编织了一张网。在这些章节里，我会继续使用"驾驭"这条主线来串联叙事，但我不会假装它完美适用于每一个案例。当隐喻变弱时，我会直接告诉你。

还有一件事。这本书不会只讲"成功的驾驭"。历史上充满了人类释放力量之后失控的故事——DDT、核武器、金融衍生品、社交媒体的信息茧房——这些"失败的驾驭"同样重要，甚至更重要。缰绳断裂的时刻，往往比缰绳套上的时刻更能告诉我们：驾驭到底意味着什么，它的代价是什么。

最后，我想明确一个立场：这本书不持技术决定论。我不相信"一旦某项技术被发明出来，后面的事情就自动展开"。事实恰恰相反——每一次成功的驾驭，背后都有制度在配合、文化在调适、组织在重构。蒸汽机需要专利法和有限责任公司才能改变世界；互联网需要开放标准和风险投资才能长成今天的样子。技术提供可能性，但人类的选择决定它最终走向哪里。

好，声明完毕。现在让我们看看这趟旅程的路线。

## 五部漫游图

我们要走的路分成五段。

**第一部 肌肉时代**（约 200 万年前 – 公元前 3500 年）：人类先驾驭自己的双手，然后驾驭火，然后驾驭别的物种，最后驾驭杠杆和轮子。能量的来源还是身体，但它已经开始被放大。

**第二部 自然力时代**（约前 3500 年 – 1760 年代）：人类开始借用地球本身的能量循环——风、水。同时，一种全新维度的"东西"被驾驭了：信息。文字、印刷、远洋航行让人类第一次跨越时空协作。

**第三部 热力时代**（1760 年代 – 1940 年代）：埋在地下几亿年的太阳能——煤和石油——被释放出来变成可控的功。蒸汽机、内燃机、电网，把人均产能提升了几十倍。

**第四部 信息时代**（1940 年代 – 2020 年代）：思维过程本身变成了可被驾驭的对象。计算机、互联网、移动与云，把"思考"和"协调"也变成了基础设施。

**第五部 智能体时代**（2020 年代起）：工具开始自己使用工具。我们驾驭的不再是某种能量或某种信号，而是"智力"本身。这一部我们还没有走完，我们正在写它。

地图铺好了，故事可以开始了。让我们回到两百多万年前，东非大裂谷的某条河滩，去看一个矮小的古人类如何用一块石头敲打另一块石头——人类的第一次驾驭，就从那里开始。
