# 第五部 智能体时代

**2020 年代 –**

> 第一次，工具自己开始使用工具。

前四部里，被驾驭的对象一直在升级：肌肉、风火水、热能、信息。但**驾驭者**始终是同一个——人。人决定何时点火、何时升帆、何时合闸、何时点击。机器无论多聪明，都在等你的下一步指令。

2020 年代开始的这一部，挑战的正是这个最后的边界。当一个大语言模型不再只是回答你的问题、而是能调用搜索引擎、读取文件、写代码、执行命令、调用其他模型，并根据中间结果**自己决定下一步该做什么**时，一种从未存在过的实体出现了：它会使用工具，而且会使用"使用工具的工具"。

这不仅仅是又一次"自动化升级"。这是驾驭对象的本体论变化——我们驾驭的不再是某种能量或某种信号，而是**智力本身**。从 75 瓦的肌肉，到几百兆瓦的数据中心，从一只手握住一块石头，到一段提示词调度一整个 Agent 集群——这条两百万年的曲线，在我们这一代人脚下迎来了它最陡的一段。

而这一部还没有结束。它正在被我们每一个人用日常的对话、提交、点击、调用，一笔一笔写出来。

这一部我们将走过三章：

- **第 14 章 大语言模型**——从 Transformer 到 ChatGPT，"语言"如何成为人与机器之间一种全新的、近乎自然的接口。
- **第 15 章 AI Agent**——当模型学会调用工具、规划步骤、自我反思，"驾驭工具的工具"诞生。Harness 一词的第三层含义在此显形。
- **第 16 章 Agent 经济学**——当一个人可以指挥成百上千个 Agent 并行工作，生产函数本身被重写，工作、组织、社会的形态将随之改变。

走完这一部，你会回到全书的最后一章，看清那根从奥杜威峡谷一路延伸到今天的缰绳——它的另一端，正在被交到你手里。
